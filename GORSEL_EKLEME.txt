===========================================
EJDERHA GÖRSELİNİ EKLEME TALİMATLARI
===========================================

Ejderha görselinizi uygulama ikonu olarak eklemek için:

1. Görseli PNG formatında hazırlayın (transparent arka planlı tercih edilir)

2. Görseli şu klasöre kopyalayın:
   YeniProje/app/src/main/res/drawable/dragon_icon_png.png

3. Dosya adı tam olarak şu şekilde olmalı:
   dragon_icon_png.png

4. Android Studio'da projeyi sync edin (File → Sync Project with Gradle Files)

5. Build edin ve test edin

ALTERNATİF YÖNTEM (Android Studio Image Asset):
-------------------------------------------------
1. Android Studio'da projeye sağ tıklayın
2. New → Image Asset seçin
3. Icon Type: Launcher Icons (Adaptive and Legacy)
4. Foreground Layer → Path → Ejderha görselinizi seçin
5. Background Layer → Color → Koyu renk (#000000 veya #1a0000)
6. Next → Finish

Bu yöntem görselinizi otomatik olarak tüm gerekli boyutlara dönüştürür.
